# PoE Price Checker v1.0.0

Initial public release.

See [README.md](README.md) for full feature list.

## Highlights

- Multi-source pricing (PoE Ninja, poe.watch, Trade API)
- Path of Building integration with build-aware evaluation
- AI-powered item analysis (Gemini, Claude, OpenAI)
- Cross-build intelligence with 20+ meta archetypes
- Unified verdict engine (FOR YOU / TO SELL / TO STASH)
- Sales tracking and loot analytics
- Modern dark theme with PoE-style aesthetics

## System Requirements

- Windows 10/11 (64-bit)
- 4GB RAM minimum
- Internet connection for pricing data

## Installation

Download the latest release and extract. No Python installation required.

For developers: See README.md for building from source.
