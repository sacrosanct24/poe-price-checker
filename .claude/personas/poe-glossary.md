# Path of Exile Glossary

A comprehensive glossary of terms, abbreviations, and slang used in Path of Exile 1 and Path of Exile 2 communities. This resource helps AI assistants and the PoE Price Checker application understand player communication.

**Last Updated**: December 8, 2025
**Update Schedule**: Alongside persona knowledge updates (every 30 days)

---

## Currency Abbreviations

### PoE1 Primary Currencies
| Abbreviation | Full Name | Usage |
|--------------|-----------|-------|
| C / c | Chaos Orb | Primary trade currency; re-rolls rare item mods |
| D / div | Divine Orb | Re-rolls mod values within tier; highest trade currency |
| Ex / exa | Exalted Orb | Adds random modifier to rare; less valuable since 3.19 |
| Alt | Orb of Alteration | Re-rolls magic item mods |
| Aug | Orb of Augmentation | Adds mod to magic item with one mod |
| Alch | Orb of Alchemy | Upgrades normal to rare |
| Scour | Orb of Scouring | Removes all mods from item |
| Chance | Orb of Chance | Upgrades normal to random rarity |
| Fuse / fusing | Orb of Fusing | Re-rolls socket links |
| Jew / jeweller | Jeweller's Orb | Re-rolls socket count |
| Chrom / chrome | Chromatic Orb | Re-rolls socket colors |
| Vaal | Vaal Orb | Corrupts item (unpredictable outcomes) |
| Mirror | Mirror of Kalandra | Duplicates rare item (extremely rare) |
| Annul | Orb of Annulment | Removes random modifier |
| Regret | Orb of Regret | Grants passive refund point |
| Blessed | Blessed Orb | Re-rolls implicit mod values |
| Regal | Regal Orb | Upgrades magic to rare (adds one mod) |
| GCP | Gemcutter's Prism | Adds quality to gems |
| Glassblower | Glassblower's Bauble | Adds quality to flasks |
| Armourer | Armourer's Scrap | Adds quality to armour |
| Whetstone | Blacksmith's Whetstone | Adds quality to weapons |

### PoE2 Currency Differences
| Currency | PoE2 Behavior | Notes |
|----------|---------------|-------|
| Exalted Orb | Primary trade currency | More common, replaces Chaos as main currency |
| Chaos Orb | Removes + adds one mod | Combines Exalt + Annul functionality |
| Divine Orb | Same as PoE1 | Still valuable for roll optimization |
| Orb of Alchemy | Scarcer than PoE1 | Save for important upgrades |

### PoE1 3.27 Keepers of the Flame Currencies
| Currency | Description |
|----------|-------------|
| Foulborn Regal Orb | Enhanced Regal with weighted high-tier mods |
| Graft Currencies | New currency type for Graft items |

## Item Terms

### Rarity
| Term | Meaning |
|------|---------|
| Normal / White | Base item, no mods |
| Magic / Blue | 1-2 mods (1 prefix, 1 suffix max) |
| Rare / Yellow | 3-6 mods (3 prefix, 3 suffix max) |
| Unique / Orange | Fixed mods with variable rolls |

### Modifier Terminology
| Term | Meaning |
|------|---------|
| Affix | General term for prefix or suffix |
| Prefix | Modifier type (up to 3 on rare items) |
| Suffix | Modifier type (up to 3 on rare items) |
| Implicit | Innate mod on base type (above separator line) |
| Explicit | Rolled mods (below separator line) |
| Tier / T1-T5 | Mod strength ranking (T1 = best) |
| ilvl | Item Level - determines max tier rollable |
| Open prefix/suffix | Empty mod slot available for crafting |
| Crafted mod | Mod added via Crafting Bench |
| Metamod | Expensive craft that protects/manipulates mods |
| Benchcraft | Crafting Bench modification |

### Item States
| Term | Meaning |
|------|---------|
| Corrupted | Modified by Vaal Orb; cannot be further modified |
| Mirrored | Copy from Mirror of Kalandra; cannot be modified |
| Fractured | One mod permanently locked |
| Synthesised | Special implicit mods from Synthesis league |
| Bricked | Ruined by bad corruption outcome |
| 6L / Six-link | 6 sockets all linked (max damage support) |
| 5L / Five-link | 5 linked sockets |
| Tabula | Tabula Rasa (6L white socket unique) |

### Influence Types (PoE1)
| Influence | Boss Source | Typical Mods |
|-----------|-------------|--------------|
| Shaper | The Shaper | Spell damage, crit, projectiles |
| Elder | The Elder | Physical DoT, minions, life |
| Crusader | Baran | Physical damage, auras |
| Hunter | Al-Hezmin | Poison, chaos damage, life |
| Redeemer | Veritania | Cold damage, crit, evasion |
| Warlord | Drox | Fire damage, fortify, endurance |
| Exarch | Searing Exarch | Fire DoT, burning |
| Eater | Eater of Worlds | Cold DoT, eldritch implicits |

### Base Types
| Term | Meaning |
|------|---------|
| Base | The underlying item type |
| High ilvl base | Base item with ilvl 86+ for T1 mods |
| Fractured base | Base with locked useful mod |
| Influenced base | Base with influence for special mods |
| Synthesised base | Base with special implicit |

## Pricing Context

### Approximate Value Tiers (PoE1 3.27)
| Tier | Value Range | Example Items |
|------|-------------|---------------|
| Vendor | < 1c | Most drops |
| Budget | 1-10c | Entry gear, basic uniques |
| Mid-tier | 10-100c | Decent rares, popular uniques |
| High-tier | 1-10 div | Good rolled uniques, crafted rares |
| Premium | 10-50 div | Well-crafted influenced items |
| Elite | 50+ div | Near-perfect items |
| Mirror-tier | 100+ div | Perfect items worth mirroring |

### PoE2 Early Access Economy Notes
- Exalted Orbs are the primary trade currency
- Divine Orbs remain valuable for optimization
- Chaos Orbs are rarer but used for targeted crafting
- Economy is still volatile during Early Access
